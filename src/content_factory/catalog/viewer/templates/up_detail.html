Почему DAG-волн и блоков УП может быть разное число

{% if quality %} DAG-волн: {{ quality.dag_wave_count or 0 }} · блоков УП: {{ quality.up_block_count or plan.summary.blocks }} {% endif %}
Волна DAG — это технический уровень зависимостей: какие skills можно изучать параллельно после закрытия пререквизитов. Блок УП — это методический модуль: несколько проектов объединяются по нагрузке, теме и проверяемому артефакту. Поэтому одна волна может быть разрезана на несколько учебных блоков, а несколько слабосвязанных волн могут попасть в один блок, если так лучше для программы.
