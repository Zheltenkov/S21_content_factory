# Основные правила редактуры

## Название Школы

Название «Школы 21» пишем:

- в кавычках и с большой буквы
- всегда с цифрой «21» и пробелом перед ней
- склоняем: «Школу 21», «Школе 21», «Школой 21»

## Названия брендов, сервисов, аббривеатур и прочего

Названия приложений, соцсетей, операционных систем, программ, редакторов, языков программирования и т. д. пишутся без кавычек латиницей, как в оригинале: Microsoft, iOS, GitHub, Phyton, CSS, SQL и пр. 
Названия технологий и позиций пишутся по-русски и с маленькой буквы: эджайл, скрам, веб, интернет, бекэнд, джун/джуниор, мидл, сеньор, тимлид, лид, скилл, софт-скиллы, хард-скиллы.  

## Другие правила редактуры

- Термин peer-to-peer пишем латиницей через дефисы без кавычек.
- Букву «ё» не используем, заменяем на «е», если это не влечет искажение смысла / двусмысленность.
- Используем кавычки «елочки», если в них заключены слова, словосочетания и предложения.
- В качестве тире используем длинное тире (—), при обозначении числовых промежутков — среднее (–).
- Постановка пробелов при перечислениях через косую черту (/):
  - если все элементы — отдельные слова, пробелы не ставим, пример: онлайн/офлайн;
  - если хотя бы один элемент — словосочетание, пробелы ставим с обеих сторон, пример: очное обучение / формат выходного дня.
