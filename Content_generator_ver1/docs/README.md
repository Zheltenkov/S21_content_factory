# Документация проекта

Этот каталог содержит только актуальные документы по текущему состоянию Content Generator. Дублирующие и не связанные с текущим production-контуром материалы вынесены из документации.

## Единый технический документ для базы знаний

- [CONTENT_GENERATOR_UNIFIED.html](CONTENT_GENERATOR_UNIFIED.html) — единый HTML-документ для публикации в конфе или базе знаний. Фокус: внутренняя ИИ-модель генерации, проверки, точечной правки и перевода. В описаниях используются русские названия вместо внутренних технических терминов.

## Для разработки и сопровождения

- [ARCHITECTURE.md](ARCHITECTURE.md) — слои системы, основные workflows, state/recovery, observability.
- [API.md](API.md) — рабочие группы REST endpoints.
- [flow_map.md](flow_map.md) — runtime-граф генерации и checkpoints.
- [AGENTS.md](AGENTS.md) — agent/LLM-компоненты и границы ответственности.
- [CRITERIA.md](CRITERIA.md) — как работает проверка README и как трактовать предупреждения.
- [TRANSLATION_MODULE.md](TRANSLATION_MODULE.md) — документный и видео-перевод.
- [TESTING.md](TESTING.md) — какие тесты запускать перед изменениями и деплоем.
- [DEPLOYMENT.md](DEPLOYMENT.md) — production-настройка и обновление сервера.

## Для пользователей

- [ИНСТРУКЦИЯ_ПО_ИСПОЛЬЗОВАНИЮ.md](ИНСТРУКЦИЯ_ПО_ИСПОЛЬЗОВАНИЮ.md) — короткая инструкция для методолога: генерация, проверка, перевод, методологический режим.
